## 什么是字节

1. 位（bit）：是计算机内部最小的储存单位，11001100是一个八位二进制数（一个2进制的1就占一位）
2. 字节（byte）是计算机中，数据处理的基本单位，习惯用大写B来表示。
3. 一（B）byte=8（bit）
4. 字符：是计算机使用字母，数字，字和符号。

1G=1024MB

1MB=1024KB

1KB=1024B

1B=1bit
